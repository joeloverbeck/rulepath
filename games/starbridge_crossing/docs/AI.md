# Starbridge Crossing AI

Starbridge Crossing ships only the Level 0 bot for this gate.

- `starbridge-crossing-l0-random-legal-v1` selects one path from Rust's public
  legal action tree using deterministic seeded randomness.
- The bot receives public state only. Starbridge has no hidden facts, private
  hands, commitments, or redacted views.
- L1/L2/L3 policies are not admitted until a competent-player note and a
  bot-strategy evidence pack exist.
- MCTS, ISMCTS, Monte Carlo search, ML, RL, and runtime LLM play are forbidden
  for public v1/v2.
